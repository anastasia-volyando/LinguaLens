# LinguaLens

LinguaLens is a multilingual educational ambiguity atlas for university linguistics students and educators. It helps learners see how syntax, semantics, and sentence structure create more than one interpretation.

## What it does

- Includes 520 linguistic entries: 130 each in English, Russian, Hebrew, and French
- Mixes ambiguous words, short phrases, and complete sentence-level analyses
- Covers attachment, scope, lexical ambiguity, coordination, reference, ellipsis, garden paths, and word formation
- Compares two readings and two structural representations for every example
- Filters the library by language, phenomenon, saved status, or search term
- Provides Hebrew transliteration and English translations for multilingual examples
- Saves examples locally in the browser for later review

## How to use it

1. Choose a language or ambiguity phenomenon, or search the example library.
2. Open an example to compare its two meanings and structural analyses.
3. Save useful examples for later review or compare the same pattern across languages.

## Known limitations

- The prototype analyzes its curated database rather than generating a complete parse for any arbitrary sentence.
- Coverage is intentionally scoped to 520 entries and does not represent every construction in the four languages.
- Explanations are educational prototype content and should receive expert review before classroom publication.
- Saved examples are stored only in the current browser and are not synchronized between devices.
- The public version does not call an external AI model because API keys cannot be stored safely in a public GitHub Pages repository.

## Privacy

Sentence analysis runs in the browser. No sentence text is uploaded or sent to an external service.
